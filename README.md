# Probate Property Finder

## Overview
The Probate Property Finder application is a comprehensive tool designed to streamline the process of finding and managing probate properties. This application offers several advanced features to enhance the user experience and effectiveness in property acquisition and management.

## Features
- **Automated Scraping**: Automatically gather data from various sources to find probate properties.
- **Contact Intelligence**: Maintain and manage contacts related to the properties efficiently.
- **Deal Scoring**: Evaluate potential deals based on set criteria and scoring mechanisms.
- **Lead Management**: Track and manage leads effectively through the sales funnel.
- **Direct Communication**: Communicate directly with leads through SMS and email functionalities.
- **Property Analytics**: Analyze property data to make informed decisions.

## Quick Start Guide
1. Clone the repository:
   ```bash
   git clone https://github.com/isaiahsintellect/Apperton.git
   cd Apperton
   ```
2. Install the required dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```

## API Endpoints Documentation
- **GET /api/properties**: Retrieve a list of probate properties.
- **POST /api/lead**: Create a new lead.
- **GET /api/lead/:id**: Retrieve a specific lead.
- **PUT /api/lead/:id**: Update an existing lead.
- **DELETE /api/lead/:id**: Delete a lead.

## Technology Stack
- **Frontend**: React
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Scraping**: Puppeteer / Cheerio
- **Authentication**: JWT (JSON Web Tokens)
